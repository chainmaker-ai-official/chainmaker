=== DRAG LINE CONNECTIVITY PROCESS ===

src/components/Designer/Blocks/DraggableBlock.jsx
  └── Contains output port (magenta bottom bar) - onMouseDown triggers handleOutputMouseDown which calls onConnectionStart

src/components/Designer/hooks/useFlowConnections.js
  └── startConnection() - captures mouse position, sets activeWire state, listens for mouseup over valid input ports (teal/green colors), dispatches addConnection to Redux

src/redux/slices/connectionsSlice.js
  └── addConnection reducer - stores { id, sourceId, targetId, sourcePort, targetPort } in connections array, marks both blocks as connected in connectedBlocks object

src/components/Designer/components/ConnectionOverlay.jsx
  └── Renders SVG bezier curves between output port (bottom) and input ports (top) using getPortPosition() to calculate coordinates from DOM

src/components/Designer/components/Workspace.jsx
  └── Container that holds ConnectionOverlay and BlockRenderer, passes connection handlers and state

src/components/Designer/components/BlockRenderer.jsx
  └── Renders appropriate block type, passes onConnectionStart/onConnectionEnd handlers from useFlowConnections hook

=== FLOW SUMMARY ===
1. Click output port → handleOutputMouseDown → onConnectionStart
2. Mouse move → activeWire tracks cursor with "rubber band" line
3. Mouse up over input port → isInputPortColor() validates target → dispatch(addConnection)
4. Redux stores connection → ConnectionOverlay renders permanent bezier curve
